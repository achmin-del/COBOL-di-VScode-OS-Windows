üéØ TUJUAN

  • Menginstall compiler GnuCOBOL melalui MSYS2 MINGW64
  • Konfigurasi environment COBOL yang berfungsi penuh
  • Integrasi dengan VS Code untuk compile dan run otomatis

üîß LANGKAH 1: INSTALASI MSYS2 & UPDATE PACKAGE

  1. Download MSYS2 dari situs resmi: https://www.msys2.org/
  2. Install dengan lokasi default (C:\msys64)
  3. Buka MSYS2 MinGW 64-bit (bukan MSYS biasa)
  4. Update package system:
pacman -Syu

Jika diminta tutup jendela, tutup dan buka kembali MSYS2 MinGW 64-bit

  1. Lanjutkan update:
pacman -Su

üîß LANGKAH 2: INSTALASI COMPILER COBOL (GnuCOBOL)

  1. Install GnuCOBOL:
pacman -S mingw-w64-x86_64-gnucobol

Ketik Y ketika diminta konfirmasi

  1. Verifikasi instalasi:
cobc -v

Seharusnya menampilkan versi GnuCOBOL (contoh: cobc (GnuCOBOL) 3.2.0)

üîß LANGKAH 3: KONFIGURASI ENVIRONMENT COBOL

Atasi error konfigurasi dengan set environment variable: 

export PATH=/mingw64/bin:$PATH
COB_CONFIG_DIR=/mingw64/share/gnucobol/config

üîß LANGKAH 4: UJI COBA PROGRAM COBOL

  1. Buat file hello.cob:
nano hello.cob
  1. Isi dengan kode:
IDENTIFICATION DIVISION.
PROGRAM-ID. HELLO.
PROCEDURE DIVISION.
    DISPLAY "HELLO, WORLD!".
    STOP RUN.
  1. Compile dan jalankan:
COB_CONFIG_DIR=/mingw64/share/gnucobol/config cobc -x -o hello.exe hello.cob
./hello.exe

Output: HELLO, WORLD!

üîß LANGKAH 5: INTEGRASI DENGAN VS CODE

A. INSTAL EXTENSION

  • Install extension COBOL (bitlang.cobol) di VS Code
  • Install extension C/C++ (Microsoft) untuk debugger

B. BUAT KONFIGURASI BUILD (.vscode/tasks.json)

{
  "version": "2.0.0",
  "tasks": [
    {
      "label": "Build COBOL",
      "type": "shell",
      "command": "bash",
      "args": [
        "-lc",
        "export PATH=/mingw64/bin:$PATH && COB_CONFIG_DIR=/mingw64/share/gnucobol/config cobc -x -o ${fileDirname}/${fileBasenameNoExtension}.exe ${file}"
      ],
      "group": {
        "kind": "build",
        "isDefault": true
      },
      "problemMatcher": []
    }
  ]
}

C. BUAT KONFIGURASI DEBUG (.vscode/launch.json)

{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Run COBOL via MSYS2 MINGW64",
      "type": "cppdbg",
      "request": "launch",
      "program": "C:/msys64/usr/bin/bash.exe",
      "args": [
        "-lc",
        "export PATH=/mingw64/bin:$PATH && cd '${fileDirname}' && ./${fileBasenameNoExtension}.exe"
      ],
      "stopAtEntry": false,
      "cwd": "${workspaceFolder}",
      "console": "integratedTerminal",
      "preLaunchTask": "Build COBOL"
    }
  ]
}

D. SETTING TERMINAL DEFAULT

  • Ctrl+Shift+P ‚Üí "Terminal: Select Default Profile"
  • Pilih MSYS2 MinGW 64-bit

üîß LANGKAH 6: PROGRAM COBOL CONTOH YANG SIAP PAKAI

1. hello.cob (Program Sederhana)

       IDENTIFICATION DIVISION.
       PROGRAM-ID. HELLO.
       PROCEDURE DIVISION.
           DISPLAY "HELLO, WORLD!".
           DISPLAY "Project: COBOL_project".
           DISPLAY "Code berhasil jalan".
           STOP RUN.

2. transaksi.cob (Program Kalkulasi dengan Format Ribuan)

       IDENTIFICATION DIVISION.
       PROGRAM-ID. TRANSAKSI.
       AUTHOR. AMIN.

       ENVIRONMENT DIVISION.
       CONFIGURATION SECTION.
       SPECIAL-NAMES.
           DECIMAL-POINT IS COMMA.

       DATA DIVISION.
       WORKING-STORAGE SECTION.

      * Input numerik
       01  HARGA-BARANG-N      PIC 9(6).
       01  JUMLAH-BELI-N       PIC 9(3).

      * Hasil perhitungan
       01  SUB-TOTAL-N         PIC 9(9).
       01  DISKON-N            PIC 9(8).
       01  TOTAL-BAYAR-N       PIC 9(9).

      * Persentase diskon
       01  PROSENTASE-DISKON   PIC 9V99 VALUE 0.10.

      * Field tampilan ribuan
       01  SUB-TOTAL           PIC ZZ.ZZZ.ZZ9.
       01  DISKON              PIC ZZ.ZZZ.ZZ9.
       01  TOTAL-BAYAR         PIC ZZ.ZZZ.ZZ9.

       PROCEDURE DIVISION.
           DISPLAY "=== PROGRAM KALKULASI TRANSAKSI ===".
           DISPLAY "Masukkan harga barang:".
           ACCEPT HARGA-BARANG-N.
           DISPLAY "Masukkan jumlah beli:".
           ACCEPT JUMLAH-BELI-N.

           COMPUTE SUB-TOTAL-N = HARGA-BARANG-N * JUMLAH-BELI-N.
           COMPUTE DISKON-N = SUB-TOTAL-N * PROSENTASE-DISKON.
           COMPUTE TOTAL-BAYAR-N = SUB-TOTAL-N - DISKON-N.

           MOVE SUB-TOTAL-N TO SUB-TOTAL.
           MOVE DISKON-N TO DISKON.
           MOVE TOTAL-BAYAR-N TO TOTAL-BAYAR.

           DISPLAY " ".
           DISPLAY "=== RINCIAN TRANSAKSI ===".
           DISPLAY "Sub Total   : " SUB-TOTAL.
           DISPLAY "Diskon (10%): " DISKON.
           DISPLAY "Total Bayar : " TOTAL-BAYAR.
           DISPLAY "==========================".
           STOP RUN.

3. biodata.cob (Program Biodata dengan Format Tanggal)

       IDENTIFICATION DIVISION.
       PROGRAM-ID. BIODATA.
       AUTHOR. AMIN.

       DATA DIVISION.
       WORKING-STORAGE SECTION.
       01  NAMA               PIC X(30).
       01  TEMPAT-LAHIR       PIC X(20).
       01  TGL-LAHIR          PIC 9(8).
       01  JK                 PIC X(1).
       01  TINGGI             PIC 9(3).
       01  TGL-FORMAT         PIC X(10).

       PROCEDURE DIVISION.
           DISPLAY "=== INPUT BIODATA ===".
           DISPLAY "Nama Lengkap:".
           ACCEPT NAMA.
           DISPLAY "Tempat Lahir:".
           ACCEPT TEMPAT-LAHIR.
           DISPLAY "Tanggal Lahir (DDMMYYYY):".
           ACCEPT TGL-LAHIR.
           DISPLAY "Jenis Kelamin (L/P):".
           ACCEPT JK.
           DISPLAY "Tinggi Badan (cm):".
           ACCEPT TINGGI.

           MOVE FUNCTION CONCAT(
                FUNCTION SUBSTRING(TGL-LAHIR FROM 1 FOR 2),
                "-", 
                FUNCTION SUBSTRING(TGL-LAHIR FROM 3 FOR 2),
                "-", 
                FUNCTION SUBSTRING(TGL-LAHIR FROM 5 FOR 4)
           )
           TO TGL-FORMAT.

           DISPLAY " ".
           DISPLAY "=== BIODATA ANDA ===".
           DISPLAY "Nama         : " NAMA.
           DISPLAY "TTL          : " FUNCTION TRIM(TEMPAT-LAHIR)
                                 " , " TGL-FORMAT.
           DISPLAY "Jenis Kelamin: " JK.
           DISPLAY "Tinggi Badan : " TINGGI " cm".
           STOP RUN.

üöÄ CARA MENJALANKAN

DI VS CODE:

  1. Buka file COBOL di editor
  2. Tekan Fn + F5 (atau F5 saja)
  3. Program akan otomatis compile dan run

DI TERMINAL MANUAL:

# Pastikan di environment MINGW64
cd /c/COBOL_project
COB_CONFIG_DIR=/mingw64/share/gnucobol/config cobc -x -o program.exe program.cob
./program.exe

‚úÖ HASIL YANG DIHARAPKAN

  • Compiler GnuCOBOL berfungsi penuh
  • Dapat compile dan run program COBOL
  • Integrasi VS Code berjalan dengan Fn+F5
  • Format angka ribuan sesuai standar Indonesia
  • Program biodata dengan format tanggal DD-MM-YYYY

üêõ TROUBLESHOOTING

  • Error "cobc: command not found": Pastikan menggunakan terminal MINGW64, bukan MSYS biasa
  • Error konfigurasi: Gunakan COB_CONFIG_DIR=/mingw64/share/gnucobol/config
  • File tidak ditemukan: Pastikan path menggunakan format Linux (/c/...) bukan Windows (C:\...)
