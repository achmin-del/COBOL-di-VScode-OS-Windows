üì¶ Proses Instalasi

1. Instalasi Visual Studio Code

Download Visual Studio Code dari website resmi (code.visualstudio.com) dan install dengan pengaturan default. Pastikan opsi "Add to PATH" tercentang saat instalasi.

2. Instalasi Extension COBOL

Buka VS Code, masuk ke Extensions (Ctrl+Shift+X), cari "bitlang.cobol" dan install. Extension ini menyediakan syntax highlighting dan code completion untuk bahasa COBOL.

3. Instalasi MSYS2 & GnuCOBOL

Download MSYS2 dari website resmi dan install. Setelah itu, buka MSYS2 MINGW64 terminal dan jalankan perintah instalasi.

BASH / SHELL 
pacman -Syu
pacman -S mingw-w64-x86_64-gnucobol
cobc --version

üíª Program COBOL

üìå Cara Menjalankan Program:

1. Tekan Ctrl + Shift + D untuk membuka "Run and Debug"
2. Pilih file yang ingin dijalankan dari dropdown
3. Tekan Fn + F5 untuk compile dan run
4. Output akan muncul di terminal

Program 1: Hello World (hello.cob)

Program dasar untuk menampilkan teks di layar.

Output Program:

===== HELLO WORLD ===== Project: COBOL_project Code Berhasil Jalan Nih Bang

Program 2: Kalkulasi Transaksi (transaksi.cob)

Program untuk menghitung total transaksi dengan format ribuan dan diskon.

Output Program (Contoh):

PROGRAM KALKULASI TRANSAKSI SEDERHANA Masukkan nama barang: Mifi Advan TIX Masukkan harga barang (tanpa titik/koma): 370000 Masukkan jumlah barang: 1 ----------------------------- Nama Barang : Mifi Advan TIX Harga Satuan : 370,000.00 Jumlah : 1.00 Total Harga : 370,000.00 Diskon 10% : 37,000.00 Total Akhir : 333,000.00 -----------------------------

Program 3: Biodata Pribadi (biodata.cob)

Program untuk input dan menampilkan biodata dengan format tanggal dan perhitungan usia.

Output Program (Contoh):

=== INPUT BIODATA === Nama Lengkap: Trex Tempat Lahir: Bumi bagian barat Tanggal Lahir (DDMMYYYY): 02052005 Jenis Kelamin (L/P): L Tinggi Badan (cm): 620cm === BIODATA ANDA === Nama : Trex TTL : Bumi bagian barat, 01-01-0001 Jenis Kelamin: L Tinggi Badan : 620 cm

üñ•Ô∏è Cara Menjalankan Program di VS Code

Langkah-Langkah Eksekusi

Step 1: Buka Run and Debug

Tekan Ctrl + Shift + D untuk membuka panel "Run and Debug" di sebelah kiri editor.

Step 2: Pilih Program

Klik dropdown di sebelah icon segitiga (play button) dan pilih program yang ingin dijalankan (Run HELLO, Run TRANSAKSI, atau Run BIODAT).

Step 3: Jalankan Program

Tekan Fn + F5 untuk mengkompilasi dan menjalankan program. Output akan muncul di integrated terminal.

üêõ Error & Solusi

Error 1: Compiler Tidak Dikenali

‚ùå Error:

cobc: command not found

Penyebab: Environment PATH tidak terkonfigurasi dengan baik.

Solusi: Set PATH untuk GnuCOBOL di MSYS2 MINGW64 terminal.

Error 2: Konfigurasi File Tidak Ditemukan

‚ùå Error:

C:\gnuCobol\config\default.conf: No such file or directory

Penyebab: Environment variable COB_CONFIG_DIR tidak tersetting.

Solusi: Set environment variable COB_CONFIG_DIR ke lokasi config GnuCOBOL.

Error 3: Integrasi VS Code Gagal

‚ùå Error:

F5 tidak bekerja atau dropdown tidak muncul

Penyebab: File konfigurasi tidak tepat atau VS Code perlu di-reload.

Solusi: Pastikan tasks.json dan launch.json sudah benar, lalu reload VS Code.
