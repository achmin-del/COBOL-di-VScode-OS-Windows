Tujuan Pembelajaran

  • Menginstall compiler GnuCOBOL melalui MSYS2 MINGW64
  • Konfigurasi environment COBOL yang berfungsi penuh
  • Integrasi dengan VS Code untuk compile dan run otomatis

üîß Langkah 1: Instalasi MSYS2 & Update Package

  1. Download MSYS2 dari situs resmi: https://www.msys2.org/
  2. Install dengan lokasi default (C:\msys64)
  3. Buka MSYS2 MinGW 64-bit (bukan MSYS biasa)
  4. Update package system:
pacman -Syu

Jika diminta tutup jendela, tutup dan buka kembali MSYS2 MinGW 64-bit 

pacman -Su

üîß Langkah 2: Instalasi Compiler COBOL (GnuCOBOL)

  1. Install GnuCOBOL:
pacman -S mingw-w64-x86_64-gnucobol

Ketik Y ketika diminta konfirmasi

  1. Verifikasi instalasi:
cobc -v

Seharusnya menampilkan versi GnuCOBOL (contoh: cobc (GnuCOBOL) 3.2.0)

üîß Langkah 3: Konfigurasi Environment COBOL

Atasi error konfigurasi dengan set environment variable: 

export PATH=/mingw64/bin:$PATH
COB_CONFIG_DIR=/mingw64/share/gnucobol/config

üîß Langkah 4: Uji Coba Program COBOL

  1. Buat file hello.cob:
nano hello.cob
  1. Isi dengan kode:
IDENTIFICATION DIVISION.
PROGRAM-ID. HELLO.
PROCEDURE DIVISION.
    DISPLAY "HELLO, WORLD!".
    STOP RUN.
  1. Compile dan jalankan:
COB_CONFIG_DIR=/mingw64/share/gnucobol/config cobc -x -o hello.exe hello.cob
./hello.exe

üîß Langkah 5: Integrasi dengan VS Code

A. Install Extension

  • Install extension COBOL (bitlang.cobol) di VS Code
  • Install extension C/C++ (Microsoft) untuk debugger

B. Buat Konfigurasi Build (.vscode/tasks.json)

{
  "version": "2.0.0",
  "tasks": [
    {
      "label": "Build COBOL",
      "type": "shell",
      "command": "bash",
      "args": [
        "-lc",
        "export PATH=/mingw64/bin:$PATH && COB_CONFIG_DIR=/mingw64/share/gnucobol/config cobc -x -o ${fileDirname}/${fileBasenameNoExtension}.exe ${file}"
      ],
      "group": {
        "kind": "build",
        "isDefault": true
      },
      "problemMatcher": []
    }
  ]
}

C. Buat Konfigurasi Debug (.vscode/launch.json)

{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Run COBOL via MSYS2 MINGW64",
      "type": "cppdbg",
      "request": "launch",
      "program": "C:/msys64/usr/bin/bash.exe",
      "args": [
        "-lc",
        "export PATH=/mingw64/bin:$PATH && cd '${fileDirname}' && ./${fileBasenameNoExtension}.exe"
      ],
      "stopAtEntry": false,
      "cwd": "${workspaceFolder}",
      "console": "integratedTerminal",
      "preLaunchTask": "Build COBOL"
    }
  ]
}

D. Setting Terminal Default

  • Ctrl+Shift+P ‚Üí "Terminal: Select Default Profile"
  • Pilih MSYS2 MinGW 64-bit

üí∞ Program Transaksi

Kalkulasi dengan format ribuan Indonesia

üë§ Program Biodata

Input dan format tanggal DD-MM-YYYY

1. hello.cob (Program Sederhana)

       IDENTIFICATION DIVISION.
       PROGRAM-ID. HELLO.
       PROCEDURE DIVISION.
           DISPLAY "HELLO, WORLD!".
           DISPLAY "Project: COBOL_project".
           DISPLAY "Code berhasil jalan".
           STOP RUN.

2. transaksi.cob (Program Kalkulasi)

       IDENTIFICATION DIVISION.
       PROGRAM-ID. TRANSAKSI.
       AUTHOR. AMIN.

       ENVIRONMENT DIVISION.
       CONFIGURATION SECTION.
       SPECIAL-NAMES.
           DECIMAL-POINT IS COMMA.

       DATA DIVISION.
       WORKING-STORAGE SECTION.

      * Input numerik
       01 WS-NUM1 PIC 9(5)V99.
       01 WS-NUM2 PIC 9(5)V99.
       01 WS-HASIL PIC 9(6)V99.
       01 WS-TAMPIL PIC Z(5)9,99.

       PROCEDURE DIVISION.
       MAIN-LOGIC.
           MOVE 12345,67 TO WS-NUM1.
           MOVE 9876,54 TO WS-NUM2.
           COMPUTE WS-HASIL = WS-NUM1 + WS-NUM2.
           MOVE WS-HASIL TO WS-TAMPIL.
           DISPLAY "HASIL PENJUMLAHAN: " WS-TAMPIL.
           STOP RUN.

3. biodata.cob (Program Input Data)

       IDENTIFICATION DIVISION.
       PROGRAM-ID. BIODATA.
       AUTHOR. AMIN.

       ENVIRONMENT DIVISION.
       INPUT-OUTPUT SECTION.
       DATA DIVISION.
       WORKING-STORAGE SECTION.
       01 WS-NAMA PIC X(20).
       01 WS-TGL-LAHIR.
          05 WS-HARI PIC 99.
          05 WS-BULAN PIC 99.
          05 WS-TAHUN PIC 9999.

       PROCEDURE DIVISION.
       MAIN-LOGIC.
           MOVE "AMIN" TO WS-NAMA.
           MOVE 15 TO WS-HARI.
           MOVE 08 TO WS-BULAN.
           MOVE 1990 TO WS-TAHUN.
           DISPLAY "NAMA: " WS-NAMA.
           DISPLAY "TANGGAL LAHIR: " 
                   WS-HARI "-" WS-BULAN "-" WS-TAHUN.
           STOP RUN.

üöÄ Cara Menjalankan Program COBOL

Langkah-langkah Menjalankan Program

  1. Buka terminal MSYS2 MinGW 64-bit
  2. Masuk ke direktori tempat file COBOL disimpan
  3. Compile program dengan perintah: cobc -x -o namafile.exe namafile.cob
  4. Jalankan program dengan: ./namafile.exe

Masalah Umum dan Solusinya

  • Error "cobc: command not found" - Pastikan GnuCOBOL terinstall dengan benar
  • Error konfigurasi - Set environment variable COB_CONFIG_DIR
  • Program tidak bisa dijalankan - Periksa sintaks dan pastikan compile berhasil
  • VS Code tidak mengenali terminal - Set default terminal ke MSYS2 MinGW 64-bit
